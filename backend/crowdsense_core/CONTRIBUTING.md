# Contributing to CrowdSense

Thank you for considering contributing to CrowdSense! Please follow these guidelines to help us maintain a collaborative and high-quality project.

## How to Contribute
- Fork the repository and create your branch from `main`.
- Write clear, concise commit messages.
- Ensure code passes all tests and lints.
- Submit a pull request with a clear description of your changes.

## Code Style
- Follow PEP8 and project linting rules.
- Add or update tests as appropriate.

## Reporting Issues
- Use the issue template for bug reports.
- Provide steps to reproduce, expected, and actual behavior.

## Code of Conduct
- Please read and follow our [Code of Conduct](CODE_OF_CONDUCT.md).

## Questions
For questions, open an issue or start a discussion.
